# Objective
1. Scrape Woolies and Coles webpages to get the list of products.
2. Get the prices of such products.
3. Reevaluate the taxonomy of products once these steps have been achieved.

# Instructions to Run
`python3 -m venv`
`python3 -m pip install -r requirements.txt`
`playwright install --deps`
`playwright install`
`python3 main.py`
